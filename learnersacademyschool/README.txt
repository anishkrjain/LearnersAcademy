It has a back-office application with a single administrator login.

How to run the application?

1. You can run the ddl script provided in the database_ddl folder. The script will create all the tables in the database. 
2. After all tables are created, enter values in the master tables anually - tb_student, tb_teacher, tb_subject, tb_class
2. Change the mySql server username and password in the SQLConnection.java class in order to connect to the database.

3. Run the index.jsp file where an admin is asked to enter the username and password.
